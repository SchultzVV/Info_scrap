🎯 ENDPOINT: endpoint_that_process_img(img_path)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Como Script:
───────────
python process_image.py minha_imagem.png
python process_image.py minha_imagem.png --modern


Como Função Python:
───────────────────
from process_image import endpoint_that_process_img

# Legacy pipeline (v1)
result = endpoint_that_process_img("image.png", use_modern=False)

# Modern pipeline (v2)
result = endpoint_that_process_img("image.png", use_modern=True)

# Resultado:
{
  "products": [
    {
      "product_name": "Café Pilão",
      "price": 13.99,
      "discount": "20%",
      "confidence": 0.95
    }
  ],
  "metadata": {
    "processing_time_seconds": 1.2,
    "num_products": 1
  }
}


Como API REST:
─────────────
# 1. Iniciar servidor
make run

# 2. Chamar endpoint
curl -X POST http://localhost:8000/analyze-flyer \
  -F "file=@image.png" \
  -H "accept: application/json"

# 3. Ver documentação
http://localhost:8000/docs
