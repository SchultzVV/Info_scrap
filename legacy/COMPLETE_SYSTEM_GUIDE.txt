📊 ARCHITECTURE COMPARISON
================================================================

┌─────────────────────────────────────────────────────────────┐
│ 1. LEGACY PIPELINE (Production-Ready)                      │
│    YOLO → OCR → LayoutLM → Product Linking                 │
│                                                              │
│    ✅ Fully tested and stable                               │
│    ✅ Modular and replaceable components                    │
│    ✅ Works out-of-the-box                                  │
│    ⚠️  Multiple components = more maintenance               │
│    ⚠️  Error propagation through stages                     │
│                                                              │
│    Files:                                                    │
│    - api.py                                                  │
│    - src/pipeline/flyer_pipeline.py                         │
│    - tests/test_pipeline.py                                 │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 2. MODERN MULTIMODAL (State-of-the-Art 2026)               │
│    image → Multimodal Transformer → structured JSON        │
│                                                              │
│    ✅ Single end-to-end model                               │
│    ✅ Better accuracy and context understanding             │
│    ✅ Faster inference                                      │
│    ✅ Easier to maintain                                    │
│    ⚠️  Requires fine-tuned model for production             │
│    ⚠️  Falls back to legacy if model not available          │
│                                                              │
│    Files:                                                    │
│    - src/serving/api_v2.py                                  │
│    - pipelines/inference_pipeline.py                        │

# Docker
docker-compose -f docker-compose-v2.yml up --build

🎯 USE CASES
================================================================

USE LEGACY PIPELINE when:
─────────────────────────
✅ Need production stability NOW
✅ Want modular, replaceable components
✅ Debugging/interpretability is important
✅ No fine-tuned multimodal model available
✅ Need to understand each stage separately

USE MODERN MULTIMODAL when:
───────────────────────────
✅ Have fine-tuned model for your data
✅ Want best-in-class accuracy
✅ Need fastest inference time
✅ Prefer single-model simplicity
✅ Following 2026 state-of-the-art
